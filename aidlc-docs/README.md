# aidlc-docs

Working artifacts generated by the **AI-DLC** workflow (see the
`aidlc-workflows/` submodule). As you run AI-DLC sessions, the methodology
writes its requirements, architecture, and construction-state documents here.

This directory is **tracked** in git, so those artifacts are versioned along
with the workspace.

> Typical structure the workflow produces:
> ```
> aidlc-docs/
> ├── aidlc-state.md                 # progress / gate checklist
> ├── inception/requirements/        # requirement docs + verification questions
> ├── construction/<unit-name>/      # per-unit design + implementation artifacts
> └── ...
> ```
